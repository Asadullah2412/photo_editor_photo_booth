# 📸 Photo Editor (OpenCV + Streamlit)

An interactive web-based photo editor built using OpenCV and Streamlit, allowing users to upload, edit, and download images in real-time with a clean and intuitive interface.

---

## 🚀 Features

### 🖼️ Image Processing

- Upload images (JPG, PNG, JPEG)
- Real-time editing with instant preview

### 🎛️ Adjustments

- Brightness control
- Contrast tuning

### 📏 Resize

- Scale image dynamically (10% – 200%)
- Maintains aspect ratio

### 🎨 Filters

- Grayscale conversion
- Gaussian Blur (adjustable)
- Sharpen filter
- Warm tone filter (Instagram-style effect)

### 🎯 Portrait Mode (Highlight Feature)

- Simulated background blur
- Keeps subject in focus using a smooth circular mask
- Adjustable blur intensity

### 📥 Export

- Download edited image in high quality

---

## 🧠 Tech Stack

- **Python**
- **OpenCV (cv2)** – Image processing
- **Streamlit** – UI and interaction
- **NumPy** – Efficient array operations
- **Pillow (PIL)** – Image handling
